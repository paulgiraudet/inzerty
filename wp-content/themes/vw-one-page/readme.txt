/* Description */
/*-----------------------------------------------------------------------------------*/

 	VW One Page is a creatively crafted, polished and eye-catching multipurpose one page WordPress theme. It is the best fit for business and corporate website, landing page, portfolio, blog, personal website, firms, enterprises, local business and many other websites for different niches of businesses. It is a responsive theme made with mobile-first approach to load perfectly on mobiles, tablets and desktops of varying sizes. It is compatible on multiple browsers and can be translated into as many as 70 different languages. This one page WordPress theme is optimized for search engines to give good traffic input to your site. With plethora of styling options of header, footer, gallery and sidebars, it promises to give your website a unique look and make it outstanding. Further, banners and sliders are provided to add beauty to it. Made from scratch, it stands firm on Bootstrap framework which makes an easy way for its usage. This one page WordPress theme gives all the power to design a website according to your specifications with its advanced customization tool. It has social media icons to make your website reach people easily. It is lightweight and compatible with third party plugins.

/* Steps to Setup Theme */
/*-----------------------------------------------------------------------------------*/

Below are the following step to setup theme static page.
=========================================================

Step 1. Add new page named as "home page" and select the template "Custom Home Page".
Step 2. Go to customizer >> static front page >> check Static page, then select the page which you have added example "home page".

For Top Header Section
======================

Step 1. Go to customizer >> VW Settings >> Topbar settings >> here you can add your contact details

For Slider Section
======================

Step 1. Add new page, add title, content and featured image and then publish it.
Step 2. Go to customizer >> VW Settings >> Slider settings >> here you can select the page which you have add for slider.

Services Section
======================

Step 1. Add new category, add new post then add title, content, assign category and featured image and then publish it.
Step 2. Go to customizer >> VW Settings >> Services >> here you can select the category which you have add for Services section.

For About Section
======================

Step 1. Add new page, add title, content and featured image and then publish it.
Step 2. Go to customizer >> VW Settings >> About >> here you can select the page which you have add for about.

For General Settings
======================

Step 1. Go to customizer >> VW Settings >> General Settings >> here you can change the layout of the blog post.
